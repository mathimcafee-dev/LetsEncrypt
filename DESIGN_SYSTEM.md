| Token | Value | Usage |

| Page background | `#f0f4fa` | Every page background |

| Amber | `#9a6400` | Warning / expiring |
| Red | `#c0392b` | Error / expired / danger |
| Ink primary | `#111111` | Headings, primary text |

| Border default | `rgba(0,119,182,0.12)` | Card edges, dividers |
